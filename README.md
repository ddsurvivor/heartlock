# HeartLock
# 心锁

### Introduction

### 玩法简介

Find the only one key to unlock the lock of heart.

找到唯一的钥匙，解开心灵之锁。

### Gameplay

### 操作方法

Mouse drag and drop the key to the lock, place the end toward keyhole.

鼠标拖动钥匙，末端对准放到锁孔上。

Press 'R' to reload the present scene.

按R重置关卡。

Press 'Q' to return to first level.

按Q回到第一关。
